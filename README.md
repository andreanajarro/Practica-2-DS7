📘 Práctica PHP – Formularios, Arreglos y XML
📌 Descripción general

En esta práctica se desarrollaron varios ejemplos en PHP con el objetivo de comprender el manejo de formularios, arreglos asociativos y lectura de archivos XML.

🧩 Ejemplo 1: Formulario con método POST

Se creó un formulario que permite al usuario ingresar su nombre y edad.
Al enviar los datos, el programa los recorre y muestra cada valor ingresado.

🔧 Ajustes realizados
Se aplicó un formato al nombre ingresado:
No importa si el usuario escribe en minúsculas o mayúsculas.
El sistema convierte el nombre a formato correcto (primera letra en mayúscula y el resto en minúscula).
Esto mejora la presentación de los datos y la experiencia del usuario.

🧩 Ejemplo 2: Arreglo asociativo

Se utilizó un arreglo asociativo para almacenar jugadores de fútbol según su posición en el campo.

El programa recorre el arreglo y muestra cada posición junto con el nombre del jugador correspondiente.

🧩 Ejemplo 3: Lectura de archivo XML

Se trabajó con un archivo XML que contiene información sobre colores y sus códigos hexadecimales.

El programa carga el archivo y muestra los datos almacenados en cada nodo, demostrando cómo PHP puede interactuar con archivos externos.
